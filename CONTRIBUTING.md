# Contributing to Cypher

Cypher is an open-source project which needs the support of open-source contributors.

Below are the instructions. We understand that there is much left to be desired, and if you see any room for improvement, please let us know. Thank you.


### Reporting An Issue

If you're about to raise an issue because you think you've found a problem or bug with Cypher, please respect the following restrictions:

- Please search for existing issues. Help us keep duplicate issues to a minimum by checking to see if someone has already reported your problem or requested your idea.

- Use the Issue Report Template below.
    ```
    1.What did you do? 

    2.What did you expect to see? 

    3.What did you see instead?
    ```
